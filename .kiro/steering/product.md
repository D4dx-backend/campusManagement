# Product Overview

d4mediaCampus is a comprehensive school management system designed for educational institutions. The system provides complete administrative and academic management capabilities for schools, with support for multi-branch operations.

## Core Functionality

- **Student Management** - Complete student lifecycle from admission to graduation
- **Staff Management** - Employee records, payroll, and HR operations  
- **Academic Structure** - Classes, divisions, departments, and designations
- **Financial Management** - Fee collection, expense tracking, and payroll processing
- **Textbook Management** - Inventory tracking and student book issuance
- **Reports & Analytics** - Comprehensive reporting across all modules
- **Multi-branch Support** - Centralized management for school chains

## User Roles

- **Super Admin** - System-wide access, branch management
- **Branch Admin** - Full access within their branch
- **Accountant** - Financial operations focus
- **Teacher** - Academic operations focus  
- **Staff** - Limited access based on permissions

## Key Features

- Role-based access control with granular permissions
- Branch-level data isolation for multi-branch schools
- JWT-based authentication with PIN login system
- Complete audit trail with activity logging
- Receipt generation and configuration
- Real-time dashboard with statistics
